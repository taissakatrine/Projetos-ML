# Projeto de Criação de Base de Dados e Treinamento da Rede YOLO
Descrição
Este projeto inclui a criação de uma base de dados e treinamento da rede YOLO, utilizando imagens do COCO e o LabelMe para rotulação.

Ferramentas Utilizadas
LabelMe: Software para rotulação de imagens

YOLO: Rede neural para detecção de objetos

COCO: Conjunto de dados de imagens rotuladas

Etapas do Projeto
Rotulação de Imagens: Utilização do LabelMe para rotular as imagens.

Preparação dos Dados: Conversão das anotações para o formato YOLO.

Treinamento da Rede: Configuração e treinamento da rede YOLO com as imagens rotuladas.

Resultados Esperados
Detecção precisa de objetos em novas imagens utilizando o modelo treinado da rede YOLO.
